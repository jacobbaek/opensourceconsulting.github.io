  • Jira 이슈 수준 보안 설정 (Configuring issue-level security)

    이슈 보안 계획에서 이슈 보안 등급을 생성하여 누가 이슈를 볼 수 있고 볼 수 없는지를 제어할 수 있습니다. 사용자나 그룹을 설정할 수 있는 보안 등급의 집합으로 이루어져 있습니다. 이슈를 생성하거나 수정할 때 해당 이슈의 보안 등급을 지정할 수 있습니다. 이렇게 하면 해당 이슈를 볼 권한이 있는 사람만이 볼 수 있게 할 수 있습니다. 이슈 보안 계획이 프로젝트와 연결 되며 해당 보안 등급은 해당 프로젝트의 이슈에 적용될 수 있습니다. 하위 작업은 부모 이슈의 보안 등급도 상속받습니다....
